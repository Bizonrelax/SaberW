Усложнённое объяснение:
1. Вставить любой текст. Поле вверху: Исходный текст.
2. Настройки: Максимальное сжатие CJK
3. Настройки: приводить к нижнему регистру
4. Сжать - кнопка
5. Копировать со словарём - кнопка
6. Если каких-то кнопок не видно - увеличить окно во весь экран.
7. Другие настройки и элементы не работают. Если работают, то не знаю почему.
8. Для сжатия текста, где регистр верхний или нижний имеет значение, снимите галочку "Приводить к нижнему регистру". Исходники Java сжимать
без этой галочки.
Проверенные нейросети.
Чаты: Дипсик, Перплексити стандарт.
Для запуска этой программы, возможно, нужна Java. Качай Java или JRE.
Программа написана на Java. Обёртка выполнена программой Launch4j.
Обёртка - это файл EXE. Исполняемый файл для Windows.
Это инструкция для пользователей, которые запустили программу
